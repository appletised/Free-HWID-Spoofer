# FREE HWID SPOOFER

Free and open-source hardware spoofer for Windows. Changes hardware identifiers to help bypass hardware bans.
This isn't my best work by far, but I am releasing this for free for everyone to use. TPM bypasses aren't included.

## Features

- **One Click Spoof** — spoofs your PC in one click
- **SMBIOS Fixer** — restores SMBIOS to semi legitimate manufacturer-correct values (ASUS, Gigabyte, MSI)
- **TPM Checker** — displays TPM endorsement key hashes
- **Serial Viewer** — shows current motherboard info, serials, and MAC addresses

## Supported Motherboards for SMBIOS Fixer

- ASUS (Intel & AMD)
- Gigabyte (Intel & AMD)
- MSI (Intel & AMD)

## Requirements

- Windows 10/11
- Admin Privileges
- CSM enabled and Secure Boot disabled in BIOS (if your motherboard spi chip is write protected)

## Usage

1. Download the latest release and extract the files
2. Run `spoof.exe` as administrator
3. Accept the disclaimer
4. Have fun
